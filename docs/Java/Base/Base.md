## 数据类型

### 包装类型

八个基本数据类型：

- boolean /1字节

- char /16字节

- byte /8字节

- short /16字节

- int /32字节

- long /64字节

- float /32字节

- double /64字节


基本类型都有对应的包装类型，基本类型与其对应的包装类型之间的赋值使用自动装箱与拆箱完成。

```java        
    Integer x = 2;     // 装箱
    
    int y = x;         // 拆箱
```
